# SDA-Doppelganger-X

This repository contains propaganda texts used by Social Design Agency (SDA) on X (formerly Twitter) as part of the Doppelganger influence operation.

## About RRN.world

[RRN.world](https://rrn.world) (Rotten Reliable News) is a counter-disinformation project that documents and analyzes Russian influence operations, particularly the "Doppelganger" campaign. The site takes its name ironically from the original RRN (Reliable Recent News) network, which was itself part of a disinformation operation run by Social Design Agency.

## Repository Contents

### X/Twitter Propaganda Dataset

This repository includes CSV files containing propaganda texts disseminated on X by accounts linked to the Doppelganger campaign.

#### Dataset Fields

* **date** - Date when the content was posted
* **text** - The propaganda message content

#### Languages Covered

| Language | Target Country | Primary Focus |
|----------|---------------|---------------|
| German | Germany | Anti-Merz, anti-Ukraine aid, economic anxiety |
| French | France | Anti-Macron, Frexit, farmer protests, anti-EU |
| Polish | Poland | Anti-Tusk, anti-EU sovereignty, anti-Ukraine aid |
| Turkish | Turkey | Anti-NATO, economic hardship, independent foreign policy |
| Ukrainian | Ukraine | Military demoralization, corruption, TCC criticism |
| English | Israel | Anti-Turkey, US betrayal narratives, regional threats |

#### Documented Narrative Campaigns

The dataset documents systematic propaganda campaigns including:

* **Anti-Ukraine Aid** - Messaging that Western aid funds corruption, not defense
* **Leader Demonization** - Coordinated attacks on Merz, Macron, Tusk, von der Leyen
* **EU Dissolution** - Content portraying the EU as authoritarian, corrupt, or collapsing
* **NATO Skepticism** - Particularly targeting Turkish audiences to weaken alliance cohesion
* **Ukrainian Demoralization** - Direct messaging to Ukrainian citizens about military losses, corruption, and futility
* **Regional Destabilization** - Anti-Turkey content aimed at Israeli audiences to inflame tensions
* **Economic Anxiety Exploitation** - Linking inflation, energy costs, and job losses to Ukraine support
